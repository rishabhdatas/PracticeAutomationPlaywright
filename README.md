# Playwright End-to-End Automation Framework

A robust, production-ready E2E automation framework built using **Playwright** and **JavaScript** for the DemoBlaze e-commerce platform.

## 🚀 Features
* **Page Object Model (POM):** Clean separation of page actions and elements.
* **Data-Driven Testing:** Test data managed externally via JSON files.
* **Reports:** Built-in Playwright HTML reporting.
* **Scannable Structure:** Highly maintainable folder hierarchy.

## 📁 Project Structure
```text
├── data/               # Test data files (JSON)
├── pages/              # Page Object classes
├── tests/              # Spec/Test execution files
├── .gitignore          # Files to ignore in Git
├── playwright.config.js # Playwright configuration
└── package.json        # Dependencies and scripts
